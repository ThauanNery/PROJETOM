🚀 Mario Jump - Evolution Edition
Este é um projeto de desenvolvimento web focado em lógica de jogos, manipulação de DOM e animações avançadas com CSS. O projeto evoluiu de um simples clone de desvio de obstáculos para um jogo de plataforma dinâmico com mecânicas de combate e ciclo ambiental.

🎮 O Jogo
O objetivo é sobreviver o maior tempo possível pulando entre plataformas sobre um mar de lava, desviando de inimigos ou eliminando-os para somar pontos de dano.

✨ Novas Funcionalidades (Versão 2.0)
Diferente da versão inicial, esta atualização traz:

Geração Procedural: As plataformas e inimigos são gerados aleatoriamente conforme você avança, tornando cada partida única.

Sistema de Combate: Implementação de projéteis (Hadouken) usando a tecla F.

Ciclo Dia/Noite Dinâmico: Um sistema de iluminação global que transita entre dia e noite a cada 30 segundos.

Ambientação Reativa: Estrelas que possuem brilho próprio e só surgem no cenário quando o céu escurece.

Câmera Dinâmica: O cenário acompanha o movimento lateral do personagem.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica e containers do jogo.

CSS3: Uso intensivo de keyframes, fixed/absolute positioning, filtros de brilho (brightness) e transições suaves.

JavaScript (Vanilla):

Lógica de colisão por proximidade (Hitbox).

Gerenciamento de estados (Game Over, Pulo, Movimento).

Manipulação de arrays para controle de múltiplos inimigos e projéteis na tela.

requestAnimationFrame para renderização fluida.

🕹️ Controles
Espaço: Pular.

Setas (Esq/Dir): Movimentar o personagem.

Tecla F: Lançar Hadouken.

Tecla R: Reiniciar o jogo após o Game Over.
