🎮 Mario Jump (Mini Game)

Este é um mini game inspirado no clássico Mario, desenvolvido com HTML, CSS e JavaScript puro.

O projeto foi baseado em um tutorial do YouTube, onde a estrutura principal do jogo já estava implementada. A partir disso, realizei melhorias próprias para praticar lógica de programação, manipulação do DOM e animações em CSS.

✨ Melhorias implementadas
🌙 Sistema de transição de dia para noite
⭐ Adição de estrelas animadas no céu
🎨 Ajustes visuais no cenário e atmosfera do jogo
🧠 Objetivo do projeto

Este projeto foi utilizado como अभ्यास prático para reforçar fundamentos de front-end, especialmente:

Manipulação do DOM com JavaScript
Animações com CSS
Posicionamento e layout com CSS
Lógica de eventos e colisão
🚀 Tecnologias utilizadas
HTML5
CSS3 (animations + keyframes)
JavaScript (vanilla)
🎯 Status

Projeto em evolução contínua para prática e aprendizado.
